== Description ==

Another facebook plugin :). Difference is the hover effect. It can be in the left or right side. Facebook plugin will be shown on hovering over the icon.

After getting huge success in [Opencart](http://www.opencart.com/index.php?route=extension/extension/info&extension_id=19473&filter_username=rosinghal), I have made this free plugin for Wordpress.

== Installation ==

1. Upload `rs-facebook-sidebar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Settings > RS Facebook Sidebar for settings.
